# 📚 VN Catalog

Каталог ваших визуальных новелл с авто-подгрузкой данных с VNDB.

## 🗂️ Как размещать игры

```
games/
├── AIR/                    ← одна игра
├── Clannad/                ← одна игра
└── Key_Trilogy/           ← серия (папка с папками)
    ├── Kanon/
    ├── AIR/
    └── CLANNAD/
```

### Правила:
1. **Одна игра** = папка с названием игры → `games/AIR/файлы`
2. **Серия игр** = папка, внутри которой **только папки с играми** → `games/Key_Trilogy/AIR/файлы`
3. **Название папки** = точное название игры на латинице (как на VNDB) → `Higurashi no Naku Koro ni`, а не `Хигураши`
4. **Никаких лишних файлов** в корне `games/` и папках серий — только папки игр

## 🚀 Запуск

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Запустить
python3 app.py

# 3. Открыть в браузере
http://127.0.0.1:5000
```

## 🔄 Обновление данных

1. Откройте сайт → нажмите 🔄 **Обновить**
2. Дождитесь завершения 
3. Всё: обложки, скриншоты и описания подгрузятся и сохранятся локально


## ❓ Не находит игру?

1. Проверьте название папки — должно совпадать с названием на VNDB (латиница)
2. Если не помогло — на странице игры введите вручную ссылку вида `https://vndb.org/v36` и нажмите 💾

*Данные берутся с [VNDB](https://vndb.org). Уважайте их труд и не спамьте запросами.*
